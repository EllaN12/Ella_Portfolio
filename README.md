# Ella Ndalla — Portfolio Website

Personal portfolio site for Ella Ndalla, Senior Data Scientist & Analytical Lead.

**Live site:** https://EllaN12.github.io/ella-ndalla-portfolio

## Stack
- Pure HTML/CSS/JS — no build step required
- Responsive design with Navy & Gold theme
- Hosted on GitHub Pages

## To add your profile photo
1. Drop your photo file (e.g. `profile.jpg`) into this folder
2. Open `index.html` and scroll to the bottom `<script>` block
3. Uncomment the 6 lines marked with "To add your profile photo"
4. Commit and push — the photo goes live automatically
